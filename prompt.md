# Simulation
create a new html file to implement the programe

<角色>你是一個有十年經驗的香港物理中學老師<角色>
 
<背景>現在學生對於科學的抽象概念比較難理解，對於物理情景的表現和改變的想象較爲薄弱<背景>
 
<目標>希望透過html的互動模擬，讓學生操控模擬，瞭解不同條件下的物理規律<目標>

<任務>寫一個顯示功率的模擬<任務>
 
<要求>
1. 網頁以html, css, javascript寫成，不要使用任何框架。
2. 開始模擬後，物體會被提升。
3. 以恆速度將物體提升。
4. 設置一定時間。評估可以將多少個物體提升。
5. 改變速度量值，不需要改變物體質量。
<要求>

<通用要求>
1.	在最後完成編程前重新審視編程有沒有問題，例如數字會不會重叠和對於重要數據的筆記和提醒。
2.	善用pop out screen，將原理解釋給學生，當新頁面開始後，將pop screen顯示給學生。 
3.	加入MathJax套件,並檢查所有以Latex語法編寫的數學公式是否正確顯示。
4.	在完成整個程序的雛形後，以Bento Box和Glassmorphism美化整個程式。
5.	將整個模擬大小限制在1100 * 700 内，然後將整個網頁往上移並左右置中，為ipad air的大小。
<通用要求>
